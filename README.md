# PAI Department CRM

Панель учёта отдела PAI (состав, еженедельные баллы, собрания, обзвон) — по структуре Excel-файла «PAI Отдел».

## Возможности

- **Состав** — просмотр сотрудников; добавление в **Настройках** (имя, фамилия, статик, должность, OOC).
- **Еженеделька** — баллы по видам активности для каждого сотрудника.
- **Собрания** — таблица явки всегда на экране (присутствовал / предупредил / примечание), без создания «нового собрания».
- **Обзвон** — 6 кандидатов + матрица вопросов с оценками 0 / 0.5 / 0.7 / 1.0 и автоматическими итогами (проход от 70%).
- **Синхронизация** — данные в `data/state.json` в репозитории; при настройке GitHub PAT изменения сохраняются для всех пользователей сайта.

## Деплой на GitHub Pages

1. Создайте репозиторий и загрузите содержимое папки `pai-department`.
2. **Settings → Pages → Source**: Deploy from branch `main`, папка `/ (root)`.
3. Откройте сайт: `https://<username>.github.io/<repo>/`
4. В **Настройки → Синхронизация (GitHub)** укажите:
   - **Owner** — ваш логин GitHub
   - **Repo** — имя репозитория
   - **GitHub Token** — [Personal Access Token](https://github.com/settings/tokens) с правом **repo** (для записи `data/state.json`)

Без токена сайт **читает** `data/state.json` из репозитория (после каждого коммита), но **не сохраняет** правки с панели — только экспорт/импорт JSON.

Токен хранится только в `localStorage` браузера (не попадает в репозиторий).

## Локальный запуск

Откройте `index.html` двойным щелчком — **данные уже встроены** в страницу (22 сотрудника).

Для загрузки `data/state.json` с диска или GitHub Pages лучше запустить локальный сервер:

```bash
cd pai-department
py -3 -m http.server 8080
```

Откройте `http://localhost:8080`

**Важно:** на GitHub загружайте **всю папку** (`index.html` + `data/state.json`), а не только HTML-файл.

## Структура

```
pai-department/
  index.html          — приложение
  data/state.json     — общие данные отдела
  README.md
```

## Обновление данных вручную

Можно редактировать `data/state.json` в GitHub и нажать **↻ Обновить** на сайте — либо настроить PAT и сохранять автоматически при любых изменениях (с интервалом опроса 30 сек).
