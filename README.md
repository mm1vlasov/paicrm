# PAI Department CRM

Панель учёта отдела PAI (состав, еженедельные баллы, собрания, обзвон) — по структуре Excel-файла «PAI Отдел».

## Возможности

- **Состав** — просмотр сотрудников; добавление в **Настройках** (имя, фамилия, статик, должность, OOC).
- **Еженеделька** — баллы по видам активности для каждого сотрудника.
- **Собрания** — таблица явки всегда на экране (присутствовал / предупредил / примечание), без создания «нового собрания».
- **Обзвон** — 6 кандидатов + матрица вопросов с оценками 0 / 0.5 / 0.7 / 1.0 и автоматическими итогами (проход от 70%).
- **Синхронизация** — данные в `data/state.json` в репозитории; при настройке GitHub PAT изменения сохраняются для всех пользователей сайта.
